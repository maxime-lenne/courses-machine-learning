Ce projet propose une introduction pédagogique aux fonctionnalités essentielles de **NumPy**, une bibliothèque fondamentale pour le calcul numérique en Python.  
Il est conçu pour comprendre les bases de la manipulation de tableaux et des opérations vectorisées avec NumPy.

## Objectifs pédagogiques

- Comprendre les concepts fondamentaux de NumPy  
- Apprendre à créer et manipuler des tableaux multidimensionnels  
- Effectuer des opérations mathématiques et statistiques sur des tableaux  

## Prérequis

- Python ≥ 3.10  
- Environnement Jupyter (ou autre IDE compatible)  
- Bibliothèques Python : NumPy  
- Connaissances de base en Python et en mathématiques  

## Installation

1. Clonez le dépôt :

   ```bash
   git clone https://github.com/Sengsathit/Teaching-NUMPY_essentiels.git
   cd Teaching-NUMPY_essentiels
   ```

2. Créez un environnement virtuel (optionnel mais recommandé) :

   ```bash
   python -m venv venv
   source venv/bin/activate  # ou .\venv\Scripts\activate sous Windows
   ```

3. Installez les dépendances :

   ```bash
   pip install -r requirements.txt
   ```

4. Lancez le notebook :

   ```bash
   jupyter notebook
   ```

## Structure du projet

- `numpy_essentiels.ipynb` : notebook principal illustrant les fonctionnalités essentielles de NumPy  
- `img/` : ressources visuelles pour la présentation  
- `requirements.txt` : liste des dépendances Python nécessaires  

## Contenu du notebook

Le notebook est structuré en plusieurs sections :

1. Introduction à NumPy et configuration de l'environnement  
2. Création et manipulation de tableaux unidimensionnels et multidimensionnels  
3. Opérations mathématiques de base et avancées sur les tableaux  
4. Fonctions statistiques et agrégation  
5. Indexation, slicing et itération sur les tableaux  
6. Différences entre les listes Python et les tableaux NumPy
